# 🤖 Haitham AI - Insurance & Transport Service Bot

Haitham AI ni WhatsApp chatbot inayosaidia wateja kupata huduma za:

- 🚗 Bima (Third Party & Comprehensive)
- 🚌 LATRA Permits
- 🆔 TIN Registration
- 🏢 TRA & BRELA Services (handover to call)

Mfumo huu unatumia Twilio WhatsApp API + Node.js backend.

---

## 🚀 Features

- 🤖 AI chatbot (rule-based)
- 📲 WhatsApp integration (Twilio)
- 💰 Automatic insurance pricing
- 🚌 Per-seat pricing (Daladala, Buses, etc.)
- 📊 Comprehensive insurance calculation
- 🔁 Customer flow automation
- 📞 TRA/BRELA handover system
